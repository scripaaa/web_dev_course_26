## Лабораторная работа 1

В 4 файлах с1-с4 реализовать функции, указанные в комментариях. В каждом файле есть тесты, которые проверяют правильность реализации функций.
Запустить файл с1_control_statement, добиться чтобы все тесты проходили, затем аналогично доработать остальные файлы.

Каждый файл посвящен определенной теме:
- c1_control_statements.rb - управляющие конструкции (условные операторы, циклы)
- c2_arrays.rb - массивы
- c3_strings.rb - строки
- c4_hashes.rb - хэши (ассоциативные массивы)

Материалы по темам можно найти в лекциях.


## Формат сдачи

Показать преподавателю. Преподаватель задаст вопросы по выполенным задачам.
